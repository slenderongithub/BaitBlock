# Glossary

Parent: [[Home]]

Mirrors `../../docs/Glossary.md`; kept here too so vault graph navigation doesn't require leaving the knowledge base.

- **Bucket** — Node's coarse risk category (`safe`/`warning`/`risky`) driving UI color.
- **Composite score** — the final 0-100 risk score. See [[Scoring-Algorithm]].
- **Extraction method** — which fallback tier successfully pulled the body text. See [[Extraction-Fallback-Chains]].
- **Semantic gap** — headline/body similarity below 0.35; means different things per backend, see [[Scoring-Algorithm]].
- **Signals** — human-readable explanations for the score, returned for explainability.
- **Verdict** — human label for the score bucket; 3-way in [[Node-Backend]], 4-way in [[Python-Backend]].

Full glossary with all terms: `../../docs/Glossary.md`.
